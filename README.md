# Adaptive Human Website

Landing page and ecosystem hub for Adaptive Human.

**Status:** In development (Conversation 1 of 4 - Foundation layer)

## Project Overview

"Beautifully broken building for beautifully broken" - Building cognitive infrastructure for people adapting to the AI age.

### Products
- **Protocol Memory** - AI memory layer (launching soon)
- **FlowScript** - Universal memory API for AI agents (in progress)
- **PLAY OS** - Play-first cognitive operating system (public beta)

## Technical Stack

- **Frontend:** Static HTML/CSS/JavaScript
- **Hosting:** Cloudflare Pages
- **Design System:** Adapted from Protocol Memory (teal + gold + purple)
- **Fonts:** Plus Jakarta Sans, Syne, Source Code Pro

## Design Philosophy

- Constraints as features (kintsugi philosophy)
- Professional + human (not corporate sterile)
- Purposeful asymmetry
- PM-level polish for MVP ’ iterate toward phillipclapham.com sophistication
